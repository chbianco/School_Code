REFRACTION NOTE:
  These parameters were estimated using a full refractive bundle
  adjustment (air->glass->water ray model). However, OpenLPT itself
  uses a standard pinhole model internally and does not model
  refraction. These files represent the best pinhole approximation
  of each camera given the refractive calibration.

  For cameras approximately perpendicular to the glass wall, this
  approximation is very accurate. For cameras at steep angles
  (>30 deg from wall normal), residual errors may be noticeable.

  For highest-accuracy 3D reconstruction, use Phase5_Reconstruction.m
  from the LPT_Calibration MATLAB package, which applies the full
  refractive ray model at every triangulation step.

WORLD COORDINATE SYSTEM:
  Origin: set during bundle adjustment (camera 1 defines the frame)
  Units: translation in mm, rotation dimensionless
  Z-axis convention: follows OpenLPT (right-hand, Z pointing into volume)
